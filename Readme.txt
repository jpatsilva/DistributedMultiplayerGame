Node.js
Installing Node (Command Line)
1. Search google for Node.js (nodes.org)
2. Download version 18.15.0 LTS
3. Run node-v18.15.0.pkg (I am using a MacOS). This installs the node.js package
4. Node.js v18.15.0 is installed in /usr/local/bin/node
5. 2pm v9.5.0 is installed in /usr/local/bin/npm
6. Make sure /usr/local/bin in in $PATH



Use NPM init to set up package.json
